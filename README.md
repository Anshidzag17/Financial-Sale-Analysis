# Financial-Sale-Analysis

This visualization is an interactive business intelligence dashboard designed to provide a comprehensive overview of a company's sales performance. The dataset behind the dashboard includes detailed sales transaction data across multiple countries, customer segments, and time periods. Key performance indicators such as total sales (17.75M), profit (2.81M), discounts (1.29M), and cost of goods sold (14.93M) are summarized at the top, offering a quick snapshot of the business's financial health.
